Week 7 Lab install instructions.
1. Ensure that you have anaconda or miniconda installed and that conda is in your path.
2. conda create -n "COMP4703Week7Lab" python=3.9
3. conda activate COMP4703Week7Lab
4. pip install jupyterlab notebook numpy matplotlib torch torchvision 
5. Run `jupyter notebook week7.ipynb` or `jupyter lab week7.ipynb`
